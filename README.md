SPY Stock Price Prediction

This project predicts SPY stock prices using regression models with engineered features
(High-Low percentage change, exponential moving averages, rolling standard deviation, and volume metrics).

Results
- Mean Absolute Error (MAE): 18.09
- Error plot shows alternating over/under predictions, suggesting model lag.

How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run `SPY_prediction.ipynb`
